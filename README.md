## 💬 Emote 시스템

채팅에 **emote 이름을 입력하면 즉시 파티클이 발동**됩니다.  

### 🧍 감정 / 반응 Emote

| 입력 | 뜻 | 파티클 |
|----|----|----|
| `angry` / `화남` | 😡 주민 분노 파티클이 생성됩니다. | `ANGRY_VILLAGER` |
| `heart` / `하트` | ❤️ 하트 파티클이 생성됩니다. | `HEART` |
| `ㅗ` / `망할` | ❤️ 하트 파티클이 생성됩니다. | `HEART` |
| `damage` / `상처` | 💔 어두운 색의 하트 파티클이 생성됩니다. | `DAMAGE_INDICATOR` |
| `critical` / `많은 상처` | 💥 어두운 색의 많은 하트 파티클이 생성됩니다. | `DAMAGE_INDICATOR (다수)` |
| `spit` / `퉤` | 🤢 라마 침 파티클이 생성됩니다. | `SPIT` |
| `no` / `하지마` | 🚫 배리어 파티클이 생성됩니다. | `BARRIER` |
| `note` / `즐` | 🎵 노트블럭의 음표 파티클이 생성됩니다. | `NOTE` |
| `rage` / `열받` | 🔥 용암 파티클이 생성됩니다. | `LAVA` |
| `tear` / `눈물` | 😢 떨어지는 물 파티클이 생성됩니다. | `FALLING_WATER` |
| `lava` / `피눈물` | 🩸 떨어지는 용암 파티클이 생성됩니다. | `FALLING_LAVA` |
| `honey` / `눈꼽` | 🍯 떨어지는 꿀 파티클이 생성됩니다. | `FALLING_HONEY` |

## 🔐 Permission 목록

서버 관리자를 위한 **우회 권한**입니다.

| Permission | 설명 |
|----------|------|
| `aimless.bypass.tabcomplete` | ⌨️ 명령어 자동완성 제한 우회 |
| `aimless.bypass.command` | ⚙️ 명령어 사용 제한 우회 |
| `aimless.bypass.use-chat` | 💬 채팅 사용 제한 우회 |
| `aimless.bypass.tablist` | 📋 탭 리스트 가리기 우회 |
